## 我完成的工作
### 1 服务器初始化与安全配置 (Server Initialization)

负责阿里云 ECS（弹性计算服务）的从零配置，构建安全可靠的宿主机环境。

- **网络与安全组配置**：合理配置阿里云安全组规则，仅开放必要的端口（如 HTTP/HTTPS、SSH 及特定服务端口），关闭不必要的外部访问。
- **基础环境搭建**：配置 SSH 密钥登录以提升安全性，并在宿主机上完成 Docker Engine 及 Docker Compose 的安装与基础调优。
- **目录结构规划**：在服务器上建立标准化的项目工作目录（包含代码目录、数据挂载目录、日志目录等）。

### 2 Docker 容器化部署 (Docker Deployment)

负责将代码及其运行环境打包，实现从本地到云端的无缝转换。

- **编排与构建**：编写及维护 `Dockerfile` 与 `docker-compose.yml`，确保前后端及中间件服务能够一键编排与启动。
- **环境隔离与网络互通**：管理 Docker Network，确保容器间通信顺畅，同时处理好从 `localhost` 到云服务器公网 IP 的映射与环境变量配置。
- **日志与监控**：配置容器日志收集规则（如限制日志文件大小），防止日志打满服务器磁盘。

### 3 数据库资源迁移 (Database Migration)

负责数据资产的安全转移与云端持久化存储。

- **数据导出与传输**：将原有环境的数据库资源（结构与数据）安全导出，并加密传输至阿里云服务器。
- **云端恢复与验证**：在云端数据库容器中完成数据导入，并执行数据完整性校验。
- **持久化管理**：通过 Docker Volumes 将数据库文件挂载到宿主机目录，避免因容器重启或销毁导致数据丢失。
