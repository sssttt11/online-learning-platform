### 3. 问题解决
- 遇到的问题: 
  1. Vercel 编译时遇到 `sh: line 1: .../vite: Permission denied` 导致构建崩溃 (Exit Code 126)。
  2. 后端初次接入真实 AI 大模型 API 时，日志报 `Insufficient Balance` 异常。
- 解决方案: 
  1. 经排查发现是 Windows 开发时将本地 `node_modules` 意外混入 Git 导致的权限丢失。通过 `git rm -r --cached` 清理暂存区，并在 Vercel 网页控制台重定向 Root Directory 至 `frontend` 纯净目录后恢复。
  2. 登录 DeepSeek 开发者平台进行财务充值激活账户，使首页 AI 向导与课程打分逻辑顺利打通。
