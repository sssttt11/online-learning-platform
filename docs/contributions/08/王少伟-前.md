## 遇到的问题和解决
1. **问题**: 初始运行测试时终端报错 `Error: Cannot find module 'vite'`。
   **解决**: 发现是由于依赖包缺失导致，在项目前端目录下重新执行 `npm install -D vite @vitejs/plugin-vue` 补全了构建依赖。
2. **问题**: 运行包含复杂样式的组件时，底层依赖抛出模块加载异常 `ERR_REQUIRE_ESM ... @csstools\css-calc\dist\index.mjs not supported`。
   **解决**: 定位到是 `jsdom` 环境底层 CSS 解析库的兼容性 Bug。通过安装更现代的 `happy-dom` (`npm install -D happy-dom`)，并在 `vite.config.js` 中将 `environment` 修改为 `happy-dom` 彻底解决了环境崩溃问题。
