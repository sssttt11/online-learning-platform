遇到的问题和解决 

1.  问题：在首页设计中，内容模块较多，导致页面布局显得拥挤 

解决：通过引入浅灰色背景块进行区域分割，并优化了课程卡片的间距（Gutter），提升了视觉层级感 

2. 问题：墨刀跳转逻辑设置较为复杂。

解决：查阅了工具教程，通过“状态栏”和“链接区域”功能实现了页面间的平滑跳转。
