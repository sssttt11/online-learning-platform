# AI 智能问答功能说明

## 一、功能概述

本功能为在线课程平台新增 **AI 智能问答服务**，基于 DeepSeek 大语言模型接口实现，为用户提供实时、智能的课程相关问题解答，提升平台交互体验与服务效率。

## 二、功能实现方式

后端采用 **Node.js + Express** 架构，通过原生 `https` 模块直接调用 DeepSeek 官方 API，不依赖第三方 SDK，保证轻量、稳定、可移植。

## 三、接口信息

- **接口地址**：`/api/ai/chat`
- **请求方式**：`POST`
- **请求类型**：`application/json`

## 四、核心业务逻辑

1. **参数校验**

   接收前端传入的 `message`，若为空或仅含空格，直接返回 `400` 错误提示。

   

2. **构建请求体**

   按照 DeepSeek API 规范组装：

   

   - 模型：`deepseek-chat`
   - 温度：`0.7`
   - 最大 tokens：`1000`
   - 用户提问内容

   

3. **发起 HTTPS 请求**

   使用 Node.js 原生 `https` 模块请求 DeepSeek 接口：

   

   - 域名：`api.deepseek.com`
   - 路径：`/v1/chat/completions`
   - 认证：`Bearer + API_KEY`（从环境变量读取）

   

4. **响应处理**

   

   - 状态码 `200`：解析 AI 回答并返回前端
   - 状态码非 `200`：返回错误信息（密钥错误、额度不足、服务异常等）
   - 网络异常：捕获并返回友好提示

   

5. **异常捕获**

   覆盖：参数错误、网络错误、API 报错、解析失败、服务器内部错误等全部场景。

   

## 五、技术特点

- 轻量：使用原生 `https`，无额外依赖
- 安全：密钥从环境变量读取，不硬编码
- 健壮：完整异常处理，接口稳定可靠
- 标准化：请求 / 响应格式统一，前端易于对接
- 可扩展：支持后续接入对话历史、多轮问答
