设计说明


配色方案

注：当前展示的为中低保真度线框原型图，以下为根据教育类网站常见风格设定的视觉规范指导颜色。

• 主色：#333333（深沉黑）—— 运用于全局顶部导航栏、重要的标题文字等，传达沉稳、专业的教育品牌形象。

• 辅助色：#F5F5F5（浅灰底色）—— 运用于页面整体背景色、卡片占位图背景（如 #E8E8E8）、模块分割线等，用于区分层级，突出前方的白色内容卡片。

• 强调色：#1890FF（科技蓝）或 #FA8C16（活力橙）—— 运用于选中状态的导航文字、价格标签（如“¥ 1000.00”）、“直播”徽标以及搜索按钮等，用于吸引用户的视觉焦点。

字体

优先使用系统默认无衬线字体，保证多平台（Windows/macOS）的清晰阅读体验 。

• 标题：PingFang SC / Microsoft YaHei, 18px - 24px, Medium/Bold（用于各个楼层大标题，如“直播课程”、“最新课程”、“热门课程”以及卡片标题如“商业插画师职业成长地图”）。

• 正文：PingFang SC / Microsoft YaHei, 12px - 14px, Regular（用于辅助说明文字，如“绘图插画 基础入门”、“1000人学过”、评分“4.0”、讲师名称等）。



核心页面

1. 首页 - 核心功能聚合展示

• 顶部导航区（Header）：包含品牌 Logo（icourse）、全局导航菜单（网站首页、全部课程、精品课堂、企业培训、高手招募、APP下载）、搜索框以及登录/注册入口。

• 核心 Banner 区：采用大图轮播形式，展示重要课程推广，左右带有切换箭头。其下方配有几个横向排列的快捷引导入口（如“高级UI设计师”等）。

• 平台优势介绍区：展示诸如“与老板高效合作”、“生活精选新课”、“人生弯道超车秘籍”等纯文本宣传语，强化平台价值。

• 直播课程楼层：4列平铺的课程卡片，卡片内含封面图、“直播”标签、标题、价格、评分及学习人数、讲师头像等信息。底部提供多选标签（如“办公效率”、“职业发展”、“编程开发”等）作为横向分类导航。

• 兴趣推荐楼层：左侧为大类目（如“设计软件”、“办公软件”），右侧为对应的4列课程卡片网格，下方附带“推荐高手”模块展示讲师信息。

• 最新/热门课程楼层：采用标准的 2行 × 4列 课程卡片瀑布流网格展示。

2. 课程详情页 - 展示单一课程的详细信息，包含课程视频预览、讲师介绍、课程大纲、用户评价及购买/加入学习按钮。
3. 全部课程页（列表/搜索页） - 提供多维度的筛选器（按类目、价格、难度等过滤），展示对应类目下更为详尽的课程列表，支持翻页。
4. 个人中心（学习控制台） - 管理用户资料、查看已购/已收藏的课程、追踪个人的学习进度和订单历史 。
5. 登录注册页 - 供新老用户进行账号注册、密码找回以及身份验证的极简表单页面。


交互说明

•  点击跳转逻辑：点击顶部导航栏的“全部课程”将跳转至课程列表页；点击首页各楼层的具体课程卡片将跳转至对应的课程详情页 。

•
悬停效果（Hover）：当鼠标悬停在任意课程卡片上时，卡片整体产生微小的 Z 轴上浮动画，并伴随边缘阴影加深（Box Shadow），给予用户明确的可点击反馈 。

•  
Banner 轮播交互：顶部的焦点大图支持自动定时轮播，鼠标悬浮时暂停，用户可点击左右两侧的箭头进行手动切换。

• 分类标签切换（Tabs）：在“直播课程”下方的分类标签（如点击“编程开发”）支持无刷新异步加载（Ajax），点击后仅该楼层的课程卡片数据进行更新替换。

•  页面滚动与加载：在长列表或瀑布流区域，页面滚动到底部时可触发“加载更多”或下拉刷新机制，加载下一页的课程数据 。
