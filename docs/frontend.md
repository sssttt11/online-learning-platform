
\# 前端开发文档（初步）



\## 一、负责模块

\- 用户注册/登录页面开发

\- 课程列表/详情页开发

\- 习题作答/错题本页面开发

\- 基础页面交互与接口对接



\## 二、技术栈

\- 开发工具：VS Code

\- 核心技术：HTML/CSS/JavaScript

\- 框架：Vue3

\- UI组件：Element Plus

\- 接口请求：Axios



\## 三、项目结构

frontend/

├── src/

│ ├── api/ # 接口请求文件

│ ├── views/ # 页面组件

│ ├── components/ # 公共组件

│ ├── App.vue # 根组件

│ └── main.js # 入口文件

└── package.json # 依赖配置





\## 四、运行步骤

1\. 进入前端目录：`cd frontend`

2\. 安装依赖：`npm install`

3\. 启动项目：`npm run dev`

4\. 访问地址：`http://localhost:5173`
